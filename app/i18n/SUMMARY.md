## 📋 Следующие шаги

### Для интеграции в существующую страницу:

1. Прочитайте `INTEGRATION_EXAMPLE.md`
2. Добавьте импорты в `app/page.tsx`
3. Добавьте состояние для языка и переводов
4. Замените хардкод текстов на переводы
5. Добавьте компонент `LanguageSwitcher` в header
6. Протестируйте переключение языков

### Для добавления новых страниц:

1. Создайте файлы переводов:
   - `app/i18n/locales/ru/[page-name].json`
   - `app/i18n/locales/kg/[page-name].json`

2. Добавьте TypeScript интерфейс в `types.ts`

3. Используйте в компоненте:
   ```typescript
   const t = await getTranslations(locale, 'page-name');
   ```

### Для добавления нового языка (например, английского):

1. Обновите `config.ts`:
   ```typescript
   locales: ['ru', 'kg', 'en']
   ```

2. Создайте папку `app/i18n/locales/en/`

3. Добавьте файлы переводов для всех страниц

4. Обновите `LanguageSwitcher.tsx`:
   ```typescript
   const languageNames: Record<Locale, string> = {
     ru: 'Русский',
     kg: 'Кыргызча',
     en: 'English',
   };
   ```
