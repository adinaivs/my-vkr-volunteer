# 🤖 AI Помощник - Документация

## Обзор

Полноценный AI помощник на базе GPT-4o mini для волонтерской платформы с минимальными затратами.

## 🎯 Возможности

### 1. **Чат с AI помощником** (`/ai-assistant`)
- Полноценный диалог с историей
- Ответы на вопросы о волонтерстве
- Отслеживание использования токенов
- Примеры вопросов

### 2. **Виджет AI помощника** (компонент)
- Встраиваемый виджет для любой страницы
- Быстрые ответы без перехода
- Минималистичный интерфейс

### 3. **API Endpoints**

#### `/api/ai/chat` - Основной чат
```typescript
POST /api/ai/chat
{
  "message": "Как стать волонтером?",
  "conversationHistory": [] // опционально
}

Response:
{
  "reply": "Ответ AI",
  "usage": {
    "prompt_tokens": 50,
    "completion_tokens": 100,
    "total_tokens": 150
  }
}
```

#### `/api/ai/suggestions` - Рекомендации проектов
```typescript
POST /api/ai/suggestions
{
  "userInterests": ["экология", "дети"],
  "userSkills": ["организация", "фотография"],
  "projectDescription": "Описание проекта..."
}

Response:
{
  "suggestion": "Краткая рекомендация",
  "usage": {...}
}
```

#### `/api/ai/analyze-project` - Анализ проекта
```typescript
POST /api/ai/analyze-project
{
  "projectTitle": "Название",
  "projectDescription": "Описание..."
}

Response:
{
  "analysis": "3 совета по улучшению",
  "usage": {...}
}
```

#### `/api/ai/generate-tasks` - Генерация задач
```typescript
POST /api/ai/generate-tasks
{
  "projectDescription": "Описание проекта...",
  "numberOfTasks": 3
}

Response:
{
  "tasks": [
    {
      "title": "Название задачи",
      "description": "Описание",
      "estimatedHours": 2
    }
  ],
  "usage": {...}
}
```

## 💰 Стоимость (GPT-4o mini)

### Цены OpenAI:
- **Входящие токены**: $0.150 / 1M токенов
- **Исходящие токены**: $0.600 / 1M токенов

### Примерные затраты:
- **1 сообщение в чате**: ~500 токенов = $0.0003 (~0.03₽)
- **1000 сообщений**: ~$0.30 (~30₽)
- **Анализ проекта**: ~300 токенов = $0.0002 (~0.02₽)
- **Генерация задач**: ~400 токенов = $0.0003 (~0.03₽)

### Оптимизация затрат:
✅ Используется самая дешевая модель GPT-4o mini
✅ Ограничение max_tokens до 500
✅ История чата ограничена 5 последними сообщениями
✅ Короткие системные промпты
✅ Оптимизированные запросы

## 🔧 Конфигурация

### Файл `lib/openai.ts`:
```typescript
export const AI_CONFIG = {
  model: 'gpt-4o-mini', // Модель
  temperature: 0.7,      // Креативность (0-1)
  max_tokens: 500,       // Макс. длина ответа
};
```

### Изменение лимитов:
- **Больше контекста**: увеличьте `slice(-5)` до `slice(-10)` в chat/route.ts
- **Длиннее ответы**: увеличьте `max_tokens` до 1000
- **Точнее ответы**: уменьшите `temperature` до 0.3

## 🐛 Решение проблем

### Ошибка "Неверный API ключ"
- Проверьте OPEN_AI в .env
- Убедитесь, что ключ начинается с `sk-proj-`
- Проверьте баланс на https://platform.openai.com/account/billing

### Ошибка "Превышен лимит"
- Подождите минуту
- Проверьте лимиты на https://platform.openai.com/account/limits
- Добавьте средства на счет

### Медленные ответы
- Уменьшите max_tokens
- Используйте более короткие промпты
- Проверьте интернет-соединение
