# Инструкция для администратора

## Вход в админ-панель

### Шаг 1: Откройте страницу входа

Перейдите по адресу: `https://volonterkr.kg/admin/login`

### Шаг 2: Введите данные

- **Email**: Ваш email администратора
- **Пароль**: Ваш пароль

Нажмите кнопку **"Войти"**

### Шаг 3: Получите код в Telegram

После ввода правильных данных:
1. Откройте Telegram
2. Найдите сообщение от бота
3. Скопируйте 6-значный код

### Шаг 4: Введите код

1. Введите код из Telegram
2. Нажмите **"Подтвердить"**

### Готово!

Вы будете перенаправлены на главную страницу админ-панели.

## Главная страница

На главной странице вы увидите:

### Статистика
- 👥 Общее количество пользователей
- 🙋 Количество волонтеров
- 🏢 Количество организаторов
- 📋 Активные проекты
- ⏳ Ожидают проверки

### Быстрые действия
- Управление пользователями
- Проверка организаторов
- Управление проектами

## Важно знать

### Безопасность
- ✅ Никогда не делитесь своим паролем
- ✅ Код из Telegram действителен только 5 минут
- ✅ Выходите из системы после работы

### Если код не пришел
1. Проверьте, что вы начали диалог с ботом (отправили `/start`)
2. Нажмите "Отправить код повторно"
3. Проверьте Telegram еще раз

### Если забыли пароль
Обратитесь к системному администратору для сброса пароля.

## Выход из системы

Нажмите кнопку **"Выйти"** в правом верхнем углу.
