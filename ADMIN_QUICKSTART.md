# Быстрый старт админ-панели

## Шаг 1: Применить миграции

```bash
npx prisma migrate deploy
```

## Шаг 2: Создать администратора

```bash
npm run create-admin
```

Введите данные администратора:
- Email
- Пароль
- Имя
- Фамилия
- Телефон
- Город

## Шаг 3: Настроить Telegram

### 3.1 Создать бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Сохраните токен бота

### 3.2 Получить User ID

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте `/start`
3. Сохраните ваш User ID

### 3.3 Сохранить настройки

```bash
npm run setup-telegram
```

Введите:
- Токен бота
- Ваш User ID

**Важно:** Отправьте `/start` вашему боту в Telegram!

## Шаг 4: Войти в админ-панель

1. Откройте браузер: `http://localhost:3000/admin/login`
2. Введите email и пароль администратора
3. Получите код в Telegram
4. Введите код на странице
5. Готово! Вы в админ-панели

## Структура админ-панели

- `/admin/login` - Вход с 2FA
- `/admin/dashboard` - Главная страница
- `/admin/users` - Управление пользователями (в разработке)
- `/admin/projects` - Управление проектами (в разработке)
- `/admin/verifications` - Проверка организаторов (в разработке)

## Безопасность

✅ Администраторы не могут заходить на страницы пользователей
✅ Пользователи не могут заходить в админ-панель
✅ Двухфакторная аутентификация через Telegram
✅ Код действителен 5 минут
✅ Автоматическое разделение ролей

## Troubleshooting

**Код не приходит?**
- Убедитесь, что вы отправили `/start` боту
- Проверьте токен бота и User ID
- Проверьте логи сервера

**Не могу войти?**
- Проверьте роль пользователя (должна быть `admin`)
- Проверьте правильность email и пароля

**Ошибка "Доступ запрещен"?**
- Только пользователи с ролью `admin` могут войти
