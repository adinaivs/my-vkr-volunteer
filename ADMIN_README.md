# Админ-панель ВолонтёрКР

## 🎯 Обзор

Полнофункциональная админ-панель с двухфакторной аутентификацией через Telegram для управления платформой волонтёрских проектов.

## ✨ Основные возможности

- ✅ **Двухфакторная аутентификация** через Telegram
- ✅ **Защищенные маршруты** с автоматическим разделением ролей
- ✅ **Статистика в реальном времени** (пользователи, проекты, верификации)
- ✅ **Современный дизайн** с темной темой и градиентами
- ✅ **Мультиязычность** (русский и кыргызский)
- ✅ **Адаптивная верстка** для всех устройств

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Применение миграций

```bash
npx prisma migrate deploy
```

### 3. Создание администратора

```bash
npm run create-admin
```

Следуйте инструкциям в консоли:
- Введите email
- Введите пароль
- Введите имя и фамилию
- Введите телефон и город

### 4. Настройка Telegram

#### 4.1 Создайте бота

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Сохраните токен бота

#### 4.2 Получите User ID

1. Откройте [@userinfobot](https://t.me/userinfobot)
2. Отправьте `/start`
3. Сохраните ваш User ID

#### 4.3 Сохраните настройки

```bash
npm run setup-telegram
```

Введите токен бота и User ID.

**⚠️ Важно:** Отправьте `/start` вашему боту в Telegram!

### 5. Запуск проекта

```bash
npm run dev
```

### 6. Вход в админ-панель

Откройте браузер: `http://localhost:3000/admin/login`

1. Введите email и пароль
2. Получите код в Telegram
3. Введите код
4. Готово!

## 🔒 Безопасность

### Разделение ролей

| Роль | `/admin/*` | `/volunteer/*` | `/organizer/*` | `/` |
|------|------------|----------------|----------------|-----|
| admin | ✅ | ❌ | ❌ | ❌ |
| volunteer | ❌ | ✅ | ❌ | ❌ |
| organizer | ❌ | ❌ | ✅ | ❌ |
| guest | ❌ | ❌ | ❌ | ✅ |

### Защита

- ✅ JWT токены с истечением через 7 дней
- ✅ HttpOnly cookies
- ✅ Хеширование паролей (bcrypt, 10 раундов)
- ✅ Коды 2FA действительны 5 минут
- ✅ Middleware проверяет роль на каждом запросе

```sql

WHERE "key" = 'ADMIN_TELEGRAM_BOT_TOKEN';
```

## 🐛 Troubleshooting

### Код не приходит в Telegram

- ✅ Проверьте токен бота в таблице `settings`
- ✅ Убедитесь, что отправили `/start` боту
- ✅ Проверьте User ID в таблице `settings`
- ✅ Проверьте логи сервера

### Ошибка "У вас нет прав администратора"

- ✅ Проверьте роль в таблице `users`
- ✅ Роль должна быть `admin`

### Код истек

- ✅ Код действителен только 5 минут
- ✅ Запросите новый код

### Не могу войти на обычные страницы

- ✅ Это нормально! Админы имеют доступ только к `/admin/*`
- ✅ Это сделано для безопасности
