# ❓ AI Помощник - Частые вопросы

## 🎯 Главный вопрос: "Он ответит на все вопросы о моем сайте?"

### ✅ ДА! Теперь AI знает о твоей платформе

После обновления AI помощник может отвечать на вопросы о:

#### 1. **Функциях платформы** ✅
- "Как зарегистрироваться?"
- "Что такое двухфакторная аутентификация?"
- "Как загрузить фото-отчет?"
- "Что делает QR-код?"

#### 2. **Процессах работы** ✅
- "Как подать заявку на проект?"
- "Как организатору создать проект?"
- "Как работает модерация?"
- "Как подтвердить выполнение задачи?"

#### 3. **Ролях пользователей** ✅
- "Кто такой организатор?"
- "Что может делать волонтер?"
- "Какие права у администратора?"

#### 4. **Реальных данных** ✅ (через `/api/ai/chat-with-context`)
- "Какие проекты сейчас активны?" → Получит из БД
- "Какие категории есть?" → Получит из БД
- "Сколько у меня заявок?" → Получит статистику

## 🔧 Как это работает?

### 1. Системный промпт (lib/openai.ts)
```typescript
export const SYSTEM_PROMPT = `
Ты - AI помощник волонтерской платформы Кыргызстана.

О ПЛАТФОРМЕ:
- Связывает волонтеров с организаторами
- Регистрация через email или Google
- Двухфакторная аутентификация для админов
- QR-коды для подтверждения участия
...
`;
```

### 2. Доступ к БД (chat-with-context)
```typescript
// AI определяет что нужны данные
if (message.includes('какие проекты')) {
  // Получает из БД
  const projects = await prisma.project.findMany({
    where: { status: 'ACTIVE' }
  });
  // Добавляет в контекст
}
```

### 3. Умный поиск (search-projects)
```typescript
// Пользователь: "Хочу помогать животным"
// AI интерпретирует → keywords: ["животные", "помощь"]
// Ищет в БД → находит проекты
// AI резюмирует → "Найдено 3 проекта..."
```

## 🚀 Как использовать?

### Вариант 1: Базовый чат (уже обновлен)
```
http://localhost:3000/ai-assistant
```
Теперь использует улучшенный промпт + доступ к БД

### Вариант 2: API напрямую
```typescript
// С контекстом БД
const res = await fetch('/api/ai/chat-with-context', {
  method: 'POST',
  body: JSON.stringify({
    message: 'Какие проекты активны?',
    userId: currentUser.id
  })
});

// Умный поиск
const res = await fetch('/api/ai/search-projects', {
  method: 'POST',
  body: JSON.stringify({
    query: 'Хочу помогать животным'
  })
});
```

## 💰 Стоимость

### Базовый чат
- ~$0.0003 за сообщение (0.03₽)

### Чат с контекстом БД
- ~$0.0005 за сообщение (0.05₽)
- Немного дороже из-за дополнительных данных

### Умный поиск
- ~$0.0006 за поиск (0.06₽)
- 2 запроса к AI (интерпретация + резюме)

**Все еще очень дешево!** 🎉
