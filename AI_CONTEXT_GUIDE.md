# 🧠 AI с контекстом платформы - Руководство

## Проблема

По умолчанию AI помощник **НЕ знает** о твоем конкретном сайте. Он отвечает только на общие вопросы о волонтерстве.

## ✅ Что сделано

### 1. Улучшен системный промпт
Теперь AI знает:
- Название и цель платформы
- Роли пользователей (волонтеры, организаторы, админы)
- Основные функции (регистрация, проекты, задачи, отчеты)
- Процесс работы (от регистрации до выполнения задач)

### 2. Создан API с доступом к БД
**`/api/ai/chat-with-context`** - умный чат с реальными данными

### 3. Создан умный поиск
**`/api/ai/search-projects`** - поиск проектов на естественном языке

## 🚀 Использование

### 1. Базовый чат (уже работает)
```typescript
// Использует улучшенный промпт
POST /api/ai/chat
{
  "message": "Как зарегистрироваться на платформе?"
}

// Ответ: "Для регистрации перейдите на страницу регистрации..."
```

### 2. Чат с контекстом БД
```typescript
POST /api/ai/chat-with-context
{
  "message": "Какие проекты сейчас доступны?",
  "userId": "user-id" // опционально
}

// Ответ включает реальные данные из БД
```

### 3. Умный поиск
```typescript
POST /api/ai/search-projects
{
  "query": "Хочу помогать животным в Бишкеке"
}

// Возвращает:
// - projects: массив найденных проектов
// - summary: краткое описание от AI
// - interpretation: как AI понял запрос
```

## 💡 Примеры интеграции

### Обновить страницу чата
```typescript
// app/ai-assistant/page.tsx
const sendMessage = async () => {

  const response = await fetch('/api/ai/chat-with-context', {

      message: input,

```

### Умный поиск на странице проектов
```typescript
// app/projects/page.tsx
const [searchQuery, setSearchQuery] = useState('');
const [aiResults, setAIResults] = useState(null);

const smartSearch = async () => {
  const res = await fetch('/api/ai/search-projects', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ query: searchQuery }),
  });

  const data = await res.json();
  setAIResults(data);
};

return (
  <div>
    <input
      value={searchQuery}
      onChange={(e) => setSearchQuery(e.target.value)}
      placeholder="Опишите что вы ищете..."
    />
    <button onClick={smartSearch}>🔍 Умный поиск</button>

    {aiResults && (
      <div>
        <p className="text-sm text-gray-600">{aiResults.summary}</p>
        {aiResults.projects.map(project => (
          <ProjectCard key={project.id} project={project} />
        ))}
      </div>
    )}
  </div>
);
```

## 💰 Стоимость

### Базовый чат
- Промпт: ~200 токенов
- Ответ: ~300 токенов
- **Итого: ~$0.0003 за сообщение**

### Чат с контекстом
- Промпт: ~400 токенов (больше из-за данных БД)
- Ответ: ~300 токенов
- **Итого: ~$0.0005 за сообщение**

### Умный поиск
- 2 запроса к AI (интерпретация + резюме)
- **Итого: ~$0.0006 за поиск**
