# ✅ База знаний обновлена!

## Проблема

AI отвечал "Извините, произошла ошибка" потому что:
1. Пытался получить данные из БД (проекты, категории)
2. Происходила ошибка при запросе к БД
3. Не хватало базовых знаний о платформе

## Решение

### 1. Отключен доступ к БД
Теперь используется простой API `/api/ai/chat` без запросов к базе данных.

### 2. Расширена база знаний
Обновлен файл `lib/openai.ts` - добавлено много информации:

#### Что теперь знает AI:

**О платформе:**
- Название и цель
- Языки (русский, кыргызский)

**О ролях:**
- Волонтеры (что могут делать)
- Организаторы (как работают)
- Администраторы (функции)

**О функциях:**
- Регистрация (email, Google)
- Проекты (категории, фильтры, карта)
- Заявки (процесс подачи)
- Задачи (назначение, выполнение)
- QR-коды (как работают)
- Фото-отчеты (загрузка, проверка)
- Волонтерская книжка (что это, как скачать)
- Статистика (часы, проекты, задачи)

**Процессы:**
- Пошаговый процесс для волонтера (10 шагов)
- Пошаговый процесс для организатора (11 шагов)

**Частые вопросы:**
- Как зарегистрироваться?
- Как найти проекты?
- Как подать заявку?
- Что такое QR-код?
- Как загрузить фото-отчет?
- Что такое волонтерская книжка?
- Как стать организатором?
- Сколько стоит?

## Где хранится база знаний?

**Файл:** `lib/openai.ts`

**Переменная:** `SYSTEM_PROMPT`

Это текстовый промпт, который отправляется AI перед каждым запросом.

## Как добавить новые знания?

1. Открой `lib/openai.ts`
2. Найди `export const SYSTEM_PROMPT`
3. Добавь информацию в нужный раздел
4. Сохрани файл
5. Перезапусти сервер (если нужно)

### Пример добавления:


## Проверь сейчас!

## Если всё равно ошибка

Проверь:
1. API ключ OpenAI в `.env` (OPEN_AI)
2. Баланс на https://platform.openai.com/account/billing
3. Логи в консоли браузера (F12)
4. Логи сервера в терминале
